# EDNA OS Website

Engineering website foundation for documenting the creation of the Orchard Operating System.

The project specifications live in `CODEX/` and remain the source of truth for architecture,
design, content, metadata, schema and accessibility decisions.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- ESLint
- Prettier

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run typecheck
npm run format:check
```

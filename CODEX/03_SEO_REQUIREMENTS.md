# Purpose

This document defines the Search Engine Optimization strategy for the EDNA OS website.

The objective is **not** short-term rankings.

The objective is long-term ownership of a new engineering category:

> **Orchard Operating System**

Every SEO decision should reinforce this objective.

---

# Primary Goal

When engineers, growers or researchers search for:

- Orchard Operating System
- Orchard Operating System Architecture
- Orchard Operating System Software
- Orchard Operating System Engineering

Google should associate those searches with:

> **EDNA OS**

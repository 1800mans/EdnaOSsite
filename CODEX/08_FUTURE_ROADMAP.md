# EDNA OS Website
# Future Roadmap
## Version 1.0

---

# Purpose

This document defines the long-term vision for the EDNA OS website.

It intentionally contains ideas that are **not yet implemented**.

The roadmap provides direction for future development while ensuring that new features remain consistent with the engineering philosophy of EDNA OS.

This document is not a release schedule.

Features should be implemented only when they support the long-term objective of establishing **Orchard Operating System** as a recognized engineering discipline.

---

# Guiding Principles

Future development should always support one or more of the following goals:

- Improve engineering documentation
- Improve educational value
- Improve technical credibility
- Improve research transparency
- Improve operational evidence
- Improve long-term maintainability

Features should never be added simply because they are fashionable.

---

# Roadmap Philosophy

The roadmap is evolutionary.

Every addition should extend the existing architecture rather than replace it.

The project should remain understandable after ten years of development.

Documentation, architecture and engineering evidence should accumulate naturally over time.

---

# Final Statement

The long-term vision of EDNA OS extends far beyond a website.

The website is intended to become the permanent public record of the creation of a new engineering discipline:

**Orchard Operating System.**

As the engineering project evolves, the website should evolve with it—documenting not only the final system, but the decisions, experiments, failures, validations and operational evidence that shaped its development.

The website itself should become an engineering artifact worthy of long-term reference.

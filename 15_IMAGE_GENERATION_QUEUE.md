# Purpose

This document manages the production of all AI-generated placeholder images used by the EDNA OS website.

It tracks the generation, approval, optimization and replacement status of every engineering image.

The image specifications remain defined in:

- 14_IMAGE_LIBRARY_GLOBAL_RULES.md
- 14_IMAGE_LIBRARY_P1.md
- 14_IMAGE_LIBRARY_P2.md
- 14_IMAGE_LIBRARY_P3.md
- 14_IMAGE_LIBRARY_P4.md
- 14_IMAGE_LIBRARY_P5.md
- 14_IMAGE_LIBRARY_P6.md

This document manages production only.

---

# Workflow

Every image follows the same lifecycle.

```

Specification

↓

Prompt Created

↓

Generated

↓

Approved

↓

Optimized

↓

Production

↓

Replaced By Real Photograph

```

---

# Status Codes

| Status | Meaning |
|---------|---------|
| TODO | Prompt not written |
| PROMPT | Prompt completed |
| GENERATED | AI image generated |
| REVIEW | Waiting for review |
| APPROVED | Approved for website |
| OPTIMIZED | Compressed and converted to WebP |
| PRODUCTION | Installed in website |
| REAL | Replaced with real orchard photograph |

# Quality Checklist

Every generated image should satisfy:

- Documentary photography
- Industrial Agriculture
- Engineering accuracy
- Realistic materials
- Natural lighting
- No AI artifacts
- No text
- No watermark
- Correct filename
- Correct aspect ratio
- WebP format

---

# Long-Term Goal

All engineering photography should eventually be replaced by original photographs from the EDNA pilot orchard while preserving filenames and references throughout the website.
